# Simulador de Caixa Eletrônico em JavaScript (ES6+)

Este é um script de lógica pura desenvolvido em JavaScript (ES6+) que simula as operações essenciais de um caixa eletrônico diretamente no ambiente de terminal/console mobile. 

O projeto foi construído de forma 100% independente, com foco em estruturas de repetição, controle de fluxo dinâmico e validações matemáticas estritas.

## 🚀 Funcionalidades

*   **Consulta de Saldo Dinâmica:** Exibe o saldo atual em tempo real refletindo qualquer operação anterior.
*   **Depósito Seguro:** Permite a entrada de valores e atualiza a persistência do saldo principal.
*   **Saque Blindado (Validação de Risco):** O sistema impede saques com valores negativos, zerados ou que ultrapassem o saldo disponível em conta.
*   **Menu Interativo Síncrono:** Controle de fluxo robusto utilizando laço de repetição contínuo até a ação explícita de saída do usuário.

## 🛠️ Tecnologias e Conceitos Aplicados

*   **JavaScript Puro (Vanilla ES6+):** Sem bibliotecas, frameworks ou interface HTML/CSS.
*   **Controle de Fluxo:** Implementação avançada de laço `while` combinado com estrutura condicional `switch/case`.
*   **Sanitização de Dados:** Conversão obrigatória de tipos de dados (`String` para `Number`) utilizando o operador unário `+` diretamente na captura de inputs.
*   **Tratamento de Escopo Léxico:** Gerenciamento preciso de variáveis locais dentro e fora de blocos síncronos.
